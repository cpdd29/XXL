# Skyscraper Assets V2

This directory stores the V2 hero-scene assets for the skyscraper page.

Current scope:

- reconstructed day/night scene plates
- manifest for Pixi scene loading
- temporary art-first runtime baseline

Why this exists:

- the V1 modular facade only validated Pixi runtime structure
- it could not approach the user's approved reference composition
- V2 switches to a `reconstructed scene plate + dynamic overlays` model first

Generation note:

- `bright.png` / `dark.png` are reference motherboards only
- runtime files in `base/` are rebuilt SVG assets
- the current plates are generated by `reception/scripts/generate-skyscraper-v2-assets.mjs`
